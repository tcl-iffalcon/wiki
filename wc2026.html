📖 REHBER

Hızlı Kurulum Adımları

🎬 Stremio Addon Kurulumu
1
Yukarıdan URL Kopyala butonuna bas.
2
web.stremio.com veya masaüstü uygulamasını aç.
3
Sol menüden Eklentiler ikonuna tıkla.
4
URL'yi yapıştır → Install'a bas.
📥 Downloader ile APK
1
TV'nde Downloader uygulamasını aç.
2
Yukarıdaki 6-7 haneli kodu veya URL'yi gir.
3
Go'ya bas — APK otomatik indirilir.
4
İndirme bittikten sonra Yükle'ye bas.
📡 TiviMate M3U Ekleme
1
TiviMate'i aç → Ayarlar → Playlist.
2
Playlist Ekle → URL seç.
3
IPTV-Org Sports M3U URL'sini yapıştır.
4
İlerleme çubuğu dolunca Spor kategorisini aç.
🔒 AWS WireGuard Kurulumu
1
AWS Console'dan EC2 → sa-east-1 (São Paulo) seç, Ubuntu 22.04 instance başlat.
2
SSH ile bağlan: sudo apt install wireguard çalıştır.
3
wg genkey ile anahtar oluştur, /etc/wireguard/wg0.conf dosyasını yapılandır.
4
Telefonunda WireGuard uygulamasına config'i ekle, Bağlan'a bas.
